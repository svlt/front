Colors

Vault uses a consistent blue color for branding and primary elements like confirmation buttons and the navbar. Other UI elements use colors from Bootstrap

Logo

Vault is currently using a simple "vault door" glyph. The logo should generally be displayed in SV blue, but can be white, black, or gray to match its surroundings. For example the site navbar is the primary color, and the logo is rendered in white on it.

Download SVG PNG PNG (Blue)

Font

The Lato font is used for headings, content, and UI elements. Headings are rendered at 500 weight, body and UI text at 400.
